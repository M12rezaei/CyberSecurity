# CMP-X305 Cyber Security Portfolio
## University of Roehampton | BSc Cyber Security

This repository contains a collection of laboratory assessments and penetration testing reports completed as part of the **CMP-X305: Cyber Security** module. 

###  Featured Labs
* **Lab 5: WordPress Penetration Testing** - Comprehensive vulnerability assessment using WPScan, API integration, and brute-force exploitation.
* **Lab 6: Web Application Vulnerabilities** - Hands-on challenges with OWASP Juice Shop.
* **Network Discovery** - Advanced Nmap scanning and service enumeration.
