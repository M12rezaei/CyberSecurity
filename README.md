# CyberSecurity
# CMP-X305 Cyber Security Portfolio
## University of Roehampton | BSc Cyber Security

This repository contains a collection of laboratory assessments and penetration testing reports completed as part of the **CMP-X305: Cyber Security** module. 

###  Featured Labs
**Lab 5: WordPress Penetration Testing** – Conducted a comprehensive vulnerability assessment using WPScan and API integration. Successfully demonstrated the full attack lifecycle, including service enumeration and administrative credential recovery via brute-force exploitation.

**Lab 7: Web Application Vulnerabilities** – Completed hands-on security challenges using the OWASP Juice Shop platform. Identified and exploited common web flaws, including Cross-Site Scripting (XSS) and SQL Injection, to understand risks to user privacy.

**Lab 8: Network Discovery & Reconnaissance** – Performed advanced network mapping using Nmap. Executed service version detection, OS fingerprinting, and vulnerability scripting (NSE) to identify exposed attack surfaces.

**Lab 9: Asset Monitoring & Vulnerability Management** – Utilized Tenable Nessus to perform enterprise-grade vulnerability research. Generated professional security audits to categorize risks by severity and provided actionable mitigation strategies for system hardening.
